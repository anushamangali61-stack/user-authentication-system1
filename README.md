# User Authentication System

A simple web-based User Authentication System developed using Python, SQL, and HTML. This project allows users to register and log in securely with database integration.

## 🚀 Features
- User Registration
- User Login
- Input Validation
- Database Integration
- Basic Error Handling

## 🛠️ Technologies Used
- Python (Flask)
- SQL (MySQL / SQLite)
- HTML
